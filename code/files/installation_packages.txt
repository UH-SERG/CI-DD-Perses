Software:

We use Python 3.7.7 and OpenJDK 1.8.0_222 with following packages:
	- scikit-learn 0.21.2
	- tensorflow 1.15.0
	- _pickle, javalang, pandas



Hardware:

We use a server with an Intel(R) Xeon(R) 2.30GHz CPU and a single NVIDIA Tesla P100 GPU with 12GB of memory to run the experiments in this study on CUDA 10.1.
